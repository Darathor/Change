Welcome to the Change UI developer guide!

You will find here the documentation about how to develop the user interface of your Change plugins.

First of all, it's important to notice that Change uses the following libs or frameworks for its backoffice interface. It's strongly recommended that you are quite comfortable with them before diving into Change UI development:

  • AngularJS (1.2)
  • jQuery (1.10)
  • Twitter Bootstrap (3)
  • FontAwesome (3.2)
  • Twig

Avoiding conflicts between Twig and Angular expressions

To avoid conflicts, Angular has been told to accept expressions enclosed in (= =), in addition to its default {{ }}.

If you have Angular expressions in Twig templates, please use (= =) so that Twig won't try to evaluate them.
